```
● Minimum digital size: 48px height
● Minimum print size: 1 inch
```

### Primary Font (Brand & Website)

● Weights: Regular, Medium, SemiBold, Bold
● Clean, modern, startup-friendly

### Secondary Font (Optional for Headlines)

**Montserrat**
● Use for emphasis or section titles

### Body Text

```
● Size: 14–16px for website
● Line-height: 1.5–1.
```
